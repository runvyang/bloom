# 学生英语水平评估追踪表（目标：PET）

**初始状态说明**
- 学生年级：四年级
- 英语学习经历：校内英语课程 + 初步课外接触
- 当前估算 CEFR 水平：Pre-A1 ~ A1（待首次评估确认）
- 目标：通过剑桥 PET（B1）考试
- 掌握程度标签：
  - 未评估（尚未进行诊断）
  - 不及格（掌握 < 60%）
  - 通过（≥ 80%）
  - 优秀（≥ 90%）
  - 精通（≥ 100%，可自由运用）

**使用方式**：根据每次评估结果，修改对应单元格的掌握程度状态。

## 首次评估说明

由于学生没有初始英语水平评估数据，**首次课程将作为全面诊断课**：
1. 快速摸底词汇量（通过图片命名、主题归类）
2. 简单对话测试口语流畅度和发音
3. 短句朗读 + 听力测试
4. 写 3-5 句自我介绍评估写作基础
5. 根据诊断结果填写上表中的初始掌握程度

诊断结果将作为所有后续教学的起点。
