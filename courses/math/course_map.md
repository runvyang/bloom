# 学生数学水平评估追踪表

**初始状态说明**  
- 学生年级：四年级（当前）  
- 奥数经历：未参加过任何奥数竞赛  
- 校内数学平均分：约 90 分  
- 掌握程度标签：  
  - 未开始  
  - 不及格（< 60）  
  - 通过（> 80）  
  - 优秀（> 90）  
  - 精通（≥ 100）  

**使用方式**：根据每次评估结果，修改对应单元格的掌握程度状态，即可直观追踪学生的知识长短板。
