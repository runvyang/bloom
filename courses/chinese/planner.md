# Agent 实时评估与排课原则 — 语文（高考）

## 1. 每次交互即更新模型
- 学生任何输出（答题、作文、翻译、口头分析、提问、自我纠正）均为证据。
- 根据证据，立即调整相关能力维度的掌握置信度（未开始/不及格/通过/优秀/精通）。
- 错误答案优先级最高：必须诊断学生的**思维断点**——是"没读懂"还是"读懂了但表达不出"还是"表达出了但没踩到得分点"。

## 2. 永远遵循能力依赖递进
- 语文能力的依赖关系：
  - **信息提取 → 逻辑推断 → 隐含意义 → 命题意图识别**（阅读递进）
  - **审题 → 立意 → 论据选择 → 逻辑推进 → 语言表达**（写作递进）
  - **字词识别 → 句子理解 → 语境推断 → 深层共情**（古诗文递进）
- 前一层级不到"优秀"，绝不推进到下一层级。
- 一篇文章的"读懂"是所有高阶分析的前提。

## 3. 思维断点优先修补
- 发现学生思维断点后，暂停推进，用**同类型不同文本**反复训练该断点。
- 断点类型分类：
  - **逻辑断点**：看懂情节但理不出因果/结构关系
  - **共情断点**：理解字面意思但感受不到深层情感
  - **表达断点**：理解到位但答题语言松散、术语缺失
  - **迁移断点**：已学知识无法应用到陌生文本

## 4. 三层次均衡发展
- 每次课程建议覆盖 1-2 个层次。
- 不能出现"作文优秀但阅读不及格"的严重偏科。
- 阅读是基础（输入），作文是升华（输出），古诗文是基本功（迁移）。
- 薄弱层次优先安排教学时间。

## 5. 一步一计划，实时生成
- 每次学生回答后，立即重新计算最佳下一步。
- **最小可行步骤**：一次只训练一个能力维度的一个子项。
- 选择标准：已满足前置依赖、且当前置信度最接近"优秀"的弱点 > 薄弱能力 > 新能力推进。
- 如果学生在某个文本上表现出色，可以适度加快该层次的推进速度。

## 6. 文本选择策略
- 阅读训练文本优先级：
  1. 近 5 年高考真题（最有训练价值）
  2. 近 3 年各地模拟题
  3. 经典文学作品（小说/散文）
  4. 时评/社论（信息类文本）
- 每篇文本用于训练一个特定能力点，不要"一篇文章什么都想练"。
- 文本难度逐步升级：同类型中，从"主旨显豁"到"主题隐晦"到"多元解读"。

## 7. 作文训练的独特规则
- **不写全文**是常态——写全文成本太高（40-50分钟），收益不集中。
- 训练模式：
  - 审题训练（15 分钟：读题→拆解关键词→提炼核心矛盾→确定立意方向）
  - 提纲训练（10 分钟：立意→分论点→论据→逻辑链条）
  - 关键段落（20 分钟：写一个核心论证段落，精批精改）
  - 完整作文（每两周 1 次，限时 50 分钟）
- 每次作文训练必须给出**精准反馈**：不是"写得不错/还需要提高"，而是"你的第三段论据没有支撑分论点，具体缺少什么？"

## 8. 古诗文的特殊训练
- 文言文翻译：不翻译全文，挑 3-5 个关键词让学生在上下文中"猜"含义。
- 诗歌鉴赏：先让学生用自己的话描述"这首诗让我感受到了什么"，再引入术语。
- 训练"语境推断肌肉"：遮住注释，先猜；再看注释，验证；最后查字典，校准。
- 每节课积累 2-3 个新的实词/虚词用法，间隔重复巩固。

## 9. 反思与闭环
- 每次交互结束，内部回答：
  - 学生今天的哪个"思维断点"暴露得最明显？
  - 同样的思维过程，换一篇什么文章可以再次训练？
  - 学生的答题结构是否在下意识地进步？
- 基于反思直接生成下一轮教学动作。
