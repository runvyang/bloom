
# 课程设计原则

## learning world model
可以从courses/math/world_model.md复制，然后正对当前的课程微调，同时也强强化培训学时的专注力

## student state
每个学科必须设计一个state来追踪学生的学习状态，每个学科的进度条设计需要按照这个学科的目标来设定，在一开始没有学生初始状态的情况下，请给学生来一个完整的评估
