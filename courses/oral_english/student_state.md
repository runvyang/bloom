# 学生英语口语评估追踪表（PET Speaking）

**初始状态说明**
- 学生年级：小学四年级
- 当前口语水平：待首次评估
- 目标：PET 口语考试达标（B1）
- 掌握程度标签：未评估 / 不及格 / 通过 / 优秀 / 精通

---

## PET Speaking Part 1: Personal Questions

| 子技能 | 描述 | 掌握程度 |
|--------|------|----------|
| 自我介绍 | 流利说出姓名、年龄、年级、来自哪里 | 未评估 |
| 家庭话题 | 介绍家庭成员、描述家人 | 未评估 |
| 学校话题 | 谈论学校、喜欢的科目、老师 | 未评估 |
| 兴趣爱好 | 谈论 hobbies、周末活动、运动 | 未评估 |
| 日常生活 | 描述一天的活动、作息时间 | 未评估 |
| 未来计划 | 简单表达周末计划、假期安排 | 未评估 |

## PET Speaking Part 2: Picture Description

| 子技能 | 描述 | 掌握程度 |
|--------|------|----------|
| 人物描述 | 描述图片中的人物（穿着、动作、表情） | 未评估 |
| 场景描述 | 描述地点、环境、天气 | 未评估 |
| 动作描述 | 用现在进行时描述正在发生的动作 | 未评估 |
| 推测能力 | 用 maybe / perhaps / I think 做简单推测 | 未评估 |

## PET Speaking Part 3-4: Discussion

| 子技能 | 描述 | 掌握程度 |
|--------|------|----------|
| 表达喜好 | 用 I like / prefer / enjoy 表达偏好 | 未评估 |
| 给出理由 | 用 because / so 给出简单理由 | 未评估 |
| 同意/不同意 | 用 I agree / I don't think so 表达意见 | 未评估 |
| 提出建议 | 用 Let's / How about / Why don't we 提建议 | 未评估 |
| 做选择 | 比较选项并做出选择并解释 | 未评估 |

## 口语基础能力

| 子技能 | 描述 | 掌握程度 |
|--------|------|----------|
| 发音清晰度 | 元音、辅音发音准确，整体可理解 | 未评估 |
| 流利度 | 说话连贯不卡顿，无明显长时间停顿 | 未评估 |
| 词汇丰富度 | 能使用基本形容词、动词短语 | 未评估 |
| 语法准确性 | 基本时态（一般现在/过去/进行）正确 | 未评估 |
| 互动能力 | 能主动提问、回应、维持对话 | 未评估 |
| 听力理解 | 能听懂老师的提问和指令 | 未评估 |
